# Отчет по лабораторной работе

## Название работы
**Анализ цветовых каналов изображений с визуализацией данных**

## Цель работы
Создание приложения с графическим интерфейсом для анализа RGB-каналов изображений и визуализации статистических данных в виде диаграмм.

## Выполненные задачи

### 1. Создание графического интерфейса
- Разработан интерфейс на базе библиотеки `tkinter`
- Добавлена информационная панель с данными студента (Капарулин Тимофей P3308)
- Реализована кнопка "Следующая" для переключения между изображениями
- Создана область отображения изображений размером 400×250 пикселей

### 2. Обработка изображений
- Реализована загрузка изображений из файлов:
  - `image1.png`
  - `image2.png`
- Конвертация изображений в RGB-формат
- Изменение размера изображений до целевого размера (400×250)

### 3. Анализ цветовых характеристик
- **Расчет средних значений RGB-каналов** с использованием `ImageStat.Stat()`
- **Подсчет доминирующих пикселей** по каналам:
  - Красный канал: пиксели, где R > G и R > B
  - Зеленый канал: пиксели, где G > R и G > B  
  - Синий канал: пиксели, где B > R и B > G

### 4. Визуализация данных
- Интеграция библиотеки `matplotlib` для создания диаграмм
- Построение столбчатых диаграмм с отображением количества пикселей по каналам
- Цветовое кодирование столбцов (красный, зеленый, синий)
- Добавление подписей значений над столбцами
- Динамическое обновление диаграмм при переключении изображений

## Технические особенности реализации

### Используемые библиотеки:
- `tkinter` - для создания графического интерфейса
- `PIL (Pillow)` - для работы с изображениями
- `matplotlib` - для создания диаграмм

### Архитектура программы:
- Класс `App` инкапсулирует всю логику приложения
- Метод `show()` отвечает за отображение изображения и анализ
- Метод `draw_bar_chart()` создает и обновляет диаграммы
- Метод `next_image()` обеспечивает переключение между изображениями

### Оптимизация производительности:
- Предотвращение накопления виджетов диаграмм при переключении изображений
- Уничтожение предыдущих диаграмм перед созданием новых

## Результаты работы

Программа успешно выполняет следующие функции:
1. **Отображение изображений** в стандартизированном размере
2. **Расчет статистики** по RGB-каналам с выводом средних значений
3. **Анализ доминирующих цветов** с подсчетом пикселей по каналам
4. **Визуализация данных** в виде интерактивных диаграмм
5. **Удобная навигация** между различными изображениями
